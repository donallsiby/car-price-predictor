This project uses Multiple Linear Regression to predict used car prices based on features like engine power, mileage, and vehicle age.

## Tech Stack
- **Python** (Pandas, Scikit-Learn)
- **Streamlit** (Web Interface)

## Data Source
The dataset is from [Kaggle](https://www.kaggle.com/datasets/manishkr1754/cardekho-used-car-data).

## How to Run
1. Install requirements: `pip install -r requirements.txt`
2. Run app: `streamlit run app.py`
